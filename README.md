# Aprender e Proteger – Navegando com Segurança

Site educativo desenvolvido em HTML, CSS e JavaScript como parte da atividade extensionista da UNINTER.

## Objetivo
Conscientizar crianças do 5º ano do ensino fundamental sobre:
- Perigos da internet
- Boas práticas de navegação
- Proteção de dados pessoais

## Tecnologias utilizadas
- HTML  
- CSS  
- JavaScript  

## Como executar
Basta abrir o arquivo index.html em qualquer navegador.
